## ğŸ“– Overview
Finding the perfect weekend getaway can be overwhelming. This project implements a **Data Engineering** solution that processes the *Top Indian Places to Visit* dataset to provide personalized travel recommendations. By calculating a multi-weighted "Rank Score," the algorithm ensures that the suggestions are not only high-quality but also geographically accessible.

## âš™ï¸ How the Ranking Algorithm Works
The engine uses a weighted scoring formula to ensure a balance between distance and quality:



1.  **Proximity Bonus:** * **+10 Points** if the destination is in the same **State** as the source city.
    * **+5 Points** if the destination is in the same **Zone** (North, South, etc.).
2.  **Quality Weight:** Google Review Rating (out of 5) is multiplied by **2.0**.
3.  **Popularity Weight:** Number of Google Reviews (in lakhs) is multiplied by **1.5**.

**Final Formula:** $$Rank Score = (Rating \times 2) + (Reviews \times 1.5) + Proximity Score$$
